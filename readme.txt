The Google+ and Disques WordPress plugin makes it easier for you to administer and customise Google+ and Disques comment from your WordPress site.

== Description ==

Simply install the plugin and follow the instructions on the Comment -> Comment Engine Pro page.

* [Live Demo](http://cepdemo.3eeweb.com/) Plugin Live Demo.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `comment-engine-pro` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Comments -> Comment Engine Pro
4. Go through the steps and hit Save Settings!
